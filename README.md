# AI Toolkit - Smart Content Processing

A modern AI-powered toolkit for content summarization, code explanation, and sentiment analysis.

## 🚀 Features

- **📝 Content Summarizer**: Summarize long texts with AI
- **💻 Code Explainer**: Get beginner-friendly code explanations
- **😊 Sentiment Analyzer**: Analyze emotions in text

## 🛠️ Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API (GPT-4o-mini)
- Hugging Face API
- React Query
- Framer Motion

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/yourusername/ai-toolkit.git
cd ai-toolkit

# Install dependencies
npm install

# Set up environment variables
cp .env.local.example .env.local
# Add your API keys to .env.local

# Run development server
npm run dev
